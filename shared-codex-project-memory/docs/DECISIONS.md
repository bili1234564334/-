## Decision Log

### D001 - Shared project memory lives in repository files

- Date: 2026-07-04
- Status: accepted
- Decision: Use `AGENTS.md` and the `docs/` folder as the shared context layer for all Codex accounts.
- Reason: Separate Codex accounts do not automatically share private chat memory. A shared repository or folder gives every account the same source of truth.
- Consequences: Each account must read the shared files at the start of work and update them at the end of work.
