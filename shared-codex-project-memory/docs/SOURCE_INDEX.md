## Status Values

- `raw`: saved but not reviewed.
- `pending summary`: needs a summary.
- `summarized`: has a summary under `docs/research/summaries/`.
- `validated`: important claims have been checked.
- `superseded`: replaced by newer material.

## Rules

- Do not paste large materials directly into chat only.
- Save or link them here so every Codex account can find them.
- If a source is important, create a short summary in `docs/research/summaries/`.
