# Shared Codex Project Memory

This repository stores the shared project context for multiple Codex accounts.

## How to Start

Every Codex account should read these files before working:

1. `AGENTS.md`
2. `docs/AI_CONTEXT.md`
3. `docs/SOURCE_INDEX.md`
4. `docs/DECISIONS.md`
5. `docs/TASK_LOG.md`

Reusable prompts are in `docs/PROMPTS.md`.

## Collaboration Rule

Do not keep important project knowledge only inside one account's chat history.

Add source materials to `docs/research/raw/`, summaries to `docs/research/summaries/`, decisions to `docs/DECISIONS.md`, and work history to `docs/TASK_LOG.md`.
